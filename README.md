# Texthue

Converts strings to hex colors.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'texthue'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install texthue

## Usage

```ruby
require 'texthue'

TextHue.convert("foobar")
=> "#3858f6"
```
